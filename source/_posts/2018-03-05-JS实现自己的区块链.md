# 区块的数据结构--链表
每个块由下面5个部分组成
{% blockquote %}
* index 区块的索引
* previousHahs 上一个区块的哈希值
* timestamp 时间戳
* data 数据，比如交易数据
* hash 当前区块的hash
{% endblockquote %}
