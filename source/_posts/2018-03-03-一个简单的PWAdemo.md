# Service Worker
{% blockquote %}
* 能够代理请求
* 能够操作浏览器缓存，通过将缓存的内容直接返回，即使离线的情况也能让页面瞬间打开。
* 可以预存储关键文件
* 可以淘汰过期的文件进行版本控制等等。
{% endblockquote %}

# ngrok
{% blockquote %}
将内网站点穿透到外网，提供https协议，方便调试
{% endblockquote %}
 

# offline.js
{% blockquote %}
很方便的监听PWA是在线状态还是离线状态，以便做对应的处理
{% endblockquote %}

# offline-plugin
{% blockquote %}
工程化的PWA工具插件
配合webpack，相当强悍，谁用谁知道~
{% endblockquote %}

# manifest
{% blockquote %}
配置manifest.json文件，将页面像原生app一样添加到桌面
{% endblockquote %}
